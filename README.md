# CSP_HWSMS — Hardware/Software Management System

A full-stack application for managing hardware and software assets.

## Repository Structure

```
CSP_HWSMS/
├── backend/    # ASP.NET Core 8 Web API (Clean Architecture)
├── frontend/   # React + TypeScript + Vite UI
└── docs/       # Project documentation (SRS, etc.)
```

## Getting Started

- **Backend** — see [`backend/README.md`](backend/README.md) for setup, entry point, and how to run the API.
- **Frontend** — see [`frontend/HWSMS_UI/README.md`](frontend/HWSMS_UI/README.md) for setup and how to run the UI.

## Tech Stack

| Layer | Technology |
|---|---|
| API | ASP.NET Core 8, Swagger / OpenAPI |
| Architecture | Clean Architecture (Domain · Application · Infrastructure · API) |
| Tests | xUnit |
| UI | React 18, TypeScript, Vite |
